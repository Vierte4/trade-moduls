Данный репозиторий хранит модули для взаимодействия с различными биржами
На данный момент в нём реализованы следующие модули:

1. ftx_seller_rest_api.py - даёт команду на покупку/продажу актива на бирже FTX с помощью протокола rest api

2. okex_parser.py - создаёт подписку на обновление валютной пары на бирже OKEX по websocket
